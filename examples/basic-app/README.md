# Basic TailwindFX Example App

## Overview
This project is a beginner-friendly example application created with TailwindFX and JavaFX.

The purpose of this example is to help new developers understand how TailwindFX components and utility-based styling can be used to build modern JavaFX interfaces.

This example includes:
- A Basic Login Form
- A Basic Dashboard Layout

These examples are designed to be simple, readable, and easy to modify for learning purposes.

---

# Features

## Basic Login Example
The login example demonstrates:
- Welcome heading
- Subtitle text
- Email input field
- Password input field
- Login button
- Forgot Password link
- Centered card-style layout

### File:
src/main/java/io/github/yasmramos/tailwindfx/examples/BasicLoginExample.java

---

## Basic Dashboard Example
The dashboard example demonstrates:
- Sidebar navigation
- Dashboard menu buttons
- Welcome header
- Statistics cards (Sales, Users, Orders)
- Analytics section
- Reports section
- Accordion / Collapse component usage

### File:
src/main/java/io/github/yasmramos/tailwindfx/examples/BasicDashboardExample.java

# How to Run

## Requirements
Before running the project, make sure you have:
- Java JDK 17 or higher
- JavaFX SDK
- IntelliJ IDEA / Eclipse / VS Code with Java support

---

## Running the Login Example
1. Open the project in your IDE
2. Navigate to:
   src/main/java/io/github/yasmramos/tailwindfx/examples/
3. Open:
   BasicLoginExample.java
4. Run the main method

---

## Running the Dashboard Example
1. Open the project in your IDE
2. Navigate to:
   src/main/java/io/github/yasmramos/tailwindfx/examples/
3. Open:
   BasicDashboardExample.java
4. Run the main method
